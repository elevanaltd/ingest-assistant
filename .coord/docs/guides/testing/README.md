# Testing Guides

This directory contains guides related to testing strategies, test coverage analysis, and validation procedures.

## Purpose
- Test strategy documentation
- Coverage analysis and reports
- Testing validation procedures
- AI structured response testing

## Current Guides
- `005-DOC-TESTING-AI-STRUCTURED.md` - AI structured response validation testing
- `TEST-COVERAGE-ACTION-FIELD.md` - Test coverage for action field implementation

## Naming Convention
- `{SEQUENCE}-DOC-TESTING-{TOPIC}.md` for numbered guides
- `TEST-COVERAGE-{FEATURE}.md` for coverage reports
- All filenames should be descriptive and use UPPER_CASE for multi-word topics
