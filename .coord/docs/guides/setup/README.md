This directory contains setup and configuration guides for the Ingest Assistant project.

- Initial project setup instructions

- Onboarding documentation

## Current Guides
- `NEW-PROJECT-SETUP-GUIDE.md` - Comprehensive new project setup guide

## Naming Convention
- `{TOPIC}-GUIDE.md` for general guides
- `SETUP-{TOPIC}.md` for specific setup procedures
- All filenames should be descriptive and use UPPER_CASE for multi-word topics
