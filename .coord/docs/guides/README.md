# Guides Directory

This directory contains implementation guides, testing procedures, and setup documentation for the Ingest Assistant project.

## Structure

### `implementation/`
Feature implementation guides, integration workflows, and POC scopes:
- POC validation criteria
- Feature implementation step-by-step guides
- Integration workflows

### `testing/`
Testing strategies, coverage analysis, and validation procedures:
- Test strategy documentation
- Coverage reports
- Validation procedures

### `setup/`
Setup and configuration guides:
- Project setup instructions
- Development environment configuration
- Onboarding documentation

## Naming Convention
- `{SEQUENCE}-DOC-{TOPIC}.md` for numbered guides
- `{TOPIC}-GUIDE.md` for general guides
- `TEST-COVERAGE-{FEATURE}.md` for coverage reports
- All filenames use UPPER_CASE for multi-word topics

## Navigation

**For implementation guidance:**
→ See `implementation/` directory

**For testing procedures:**
→ See `testing/` directory

**For project setup:**
→ See `setup/` directory
