# Lessons Learned Directory

This directory contains postmortems, retrospectives, and lessons learned from the Ingest Assistant project.

## Purpose
- Postmortem analyses
- Retrospective summaries
- Technical debt reviews
- Architectural decision retrospectives
- Process improvement insights

## Naming Convention
- `{SEQUENCE}-LESSON-{TOPIC}.md`
- `{DATE}-POSTMORTEM-{INCIDENT}.md`
- Example: `001-LESSON-VIDEO-TRANSCODING-APPROACH.md`

## Guidelines
- Document what went well and what didn't
- Include context for future developers
- Link to relevant ADRs or reports
- Suggest actionable improvements
